Langkah-langkah menggunakan github

*membuat akun

Langkah pertama kilk github, setelah itu buatlah akun Anda. Setelah selesai, langsung masuk ke halaman utama dan klik tombol “+” untuk membuat repositori baru.

*membuat repository online

Setelah membuat akun, langkah berikutnya adalah membuat repository baru. apabila sudah membuat repository baru selanjutnya klik "create repository" maka halaman selanjutnya akan menampilkan hasilnya. langkah berikutnya adalah mengunggah project ke repository online.

*mengunggah folder

Sebelum melakukan pengungguhan,pastikan bahwa project yang akan diunggah sudah tersedia git. Setelah itu, bukalah folder project dengan file eksplorer kemudian klik kanan “open terminal/cmder here”.

Dalam melakukan pengunggahan, ada beberapa perintah dasar yang digunakan, yaitu : git init, git add, git commit -m, git remote add origin, git push -u origin master
